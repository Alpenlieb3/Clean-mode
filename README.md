# 🎬 YouTube Clean Mode

A lightweight bookmarklet that removes YouTube UI elements for a clean, distraction-free viewing experience while keeping the video playing normally.


## 😤 Problem

YouTube adds many interface elements that reduce focus while watching videos:

* Titles and metadata overlays
* Playback controls and progress bar
* End screens and recommendations
* Tooltips and popups
* UI that reappears on mouse movement

Even in fullscreen, these elements break immersion.


## 💡 Solution

This tool hides all YouTube interface layers using a small CSS injection script, turning YouTube into a clean cinema-style video player.


## ⚙️ How It Works

* Injects CSS into the page
* Hides YouTube UI elements (controls, overlays, suggestions)
* Keeps only the video visible
* Forces cursor visibility
* Does not affect video playback


## 🧩 Features

* One-click activation (bookmarklet)
* No installation required
* Video plays normally
* Removes all UI distractions
* Cursor always visible
* Toggle on/off anytime


## 🌐 Compatibility

### ✅ Works on:

* YouTube (desktop)
* YouTube Music (partial)
* HTML5 video players
* Embedded YouTube videos

### ⚠️ Limited support:

* Fullscreen mode behavior may vary
* Sandboxed embedded players

### ❌ Not supported:

* Mobile apps
* DRM-protected streaming platforms
* Browsers blocking bookmarklets


## 🔒 Safety

* Runs only in your browser
* No data collection
* No external requests
* No tracking or cookies
* Fully reversible

Note: YouTube updates may occasionally require selector updates.


## 🧠 In Short

A simple tool that removes YouTube UI clutter and creates a clean, distraction-free viewing experience without affecting playback.
