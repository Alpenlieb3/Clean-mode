# 🎬 Clean Mode

Turn YouTube into a distraction-free video player by removing all UI elements while keeping the video running normally.

---

## 🚀 Live Demo

👉 https://alpenlieb3.github.io/Clean-mode/

---

## 📌 How to use

1. Open the link above  
2. Drag the red button to your bookmarks bar  
3. Go to YouTube  
4. Click the bookmark to activate Clean Mode  

---

## ⚠️ Important

- Bookmarks bar must be visible  
- Works only on desktop browsers  
- Click bookmark while on YouTube  

---

## 🎯 Features

- Removes YouTube UI distractions  
- Keeps video playback normal  
- One-click toggle via bookmark  
- No installation required
